如果你能看到这篇文章，说明这个 Hexo 站点已经可以正常生成页面了。

接下来通常会做三件事：

1. 修改站点配置，比如博客标题、作者名、 GitHub 仓库地址。
2. 选择一个你喜欢的主题，或者继续定制默认主题。
3. 把 `source/_posts` 里的文章持续写起来，再发布到 GitHub Pages。

本项目已经预装了 `hexo-deployer-git`，后续只要配置好仓库地址，就可以直接部署。
